
# Matplotlib and Seaborn Practice: 
This project is built using the python pathlib, pandas, matplotlib and seaborn modules.

## Dataset
Public datasets downloaded from kaggle:
- Churn.csv:
	7 Features (CustomerID, Age, MonthlyBill, DataUsageGB, ContractType, TenureMonths, Churn)
	25 Records (Rows)
- Electricity.csv
	2 Features (Date, IPG2211A2N)
	357 Records
