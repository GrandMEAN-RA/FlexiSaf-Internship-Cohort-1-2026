
# Simulation: Tic-Tac-toe
This project is built using the python random module.

## How the AI learns (simple explanation)
- The AI stores board states that led to wins or losses
- It prefers moves that previously led to wins
- It avoids moves that previously led to losses
- Over repeated games, it improves

## Features
- Function-based tasks delegation
- AI mode training mode
- Two players mode
