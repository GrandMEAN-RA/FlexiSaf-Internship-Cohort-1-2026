
# Exploratory Data Analysis: 
This project is built using the python pathlib, pandas and numpy modules.

## Dataset
Public dataset with:
- 7 Features (CustomerID, Age, MonthlyBill, DataUsageGB, ContractType, TenureMonths, Churn)
- 25 Records (Rows)

## Features
- Basic i/o file operations
- Basic use cases of:
	- Lists
	- Dictionaries
	- Dataframes
	- Elemetary Statistics
- Data Exploration with pandas and numpy
	- Mean
	- Median
	- Mode
	- Standard Deviation
	- Correlation matrix
